# Task 17: Edición de Perfil de Usuario
**Asignado a:** Frontend Specialist Agent / Backend Agent

## Objetivo
Permitir que el usuario pueda editar y actualizar su nombre cuando acceda a la configuración de su perfil.

## Requerimientos Funcionales
1. **Interfaz de Edición:** Al seleccionar la opción o el botón "Edit" (Editar) en el perfil/dashboard del usuario, el campo de nombre (Name) debe habilitarse para edición o abrirse en un modal.
2. **Actualización de Datos:** Añadir un botón para "Guardar Cambios" que permita enviar el nuevo nombre al backend (Supabase).
3. **Validación:** El campo no debe poder ser guardado si está vacío.
4. **Respuesta Visual (UX):** Mostrar una notificación de éxito o un *toast* al guardar correctamente el nombre, o un error si el cambio falla.
