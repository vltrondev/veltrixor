# Tarea 3.1: Vistas de Autenticación con Supabase Auth

**Asignado a:** Agente Especialista en Frontend (React & Vite)
**Estado:** Pendiente

## Descripción de la Tarea
Debes implementar las pantallas de **Registro** e **Inicio de Sesión (Login)** para Veltrixor. Es fundamental que tengas en cuenta que toda la lógica de validación, creación de usuarios y sesiones será gestionada por **Supabase Auth**.

### Requerimientos Estrictos:
1. **No Backend Extra**: No construyas lógica compleja de JWT, ni bases de datos propias. Prepara los componentes para integrarse con `@supabase/supabase-js`.
2. **Formularios Controlados (`react`)**:
   - Campos de Email y Contraseña.
   - Manejo básico de estados de "Cargando" (Loading) y visualización de "Errores".
3. **Diseño Premium**:
   - Usa un esquema oscuro corporativo (Dark Mode).
   - Componentes centralizados flotantes usando *Glassmorphism* (fondos translúcidos y bordes difuminados).
   - Botones atractivos con animaciones sutiles (hover, active).
4. **Archivos a Crear/Modificar**:
   - `src/pages/Login.jsx`
   - `src/pages/Register.jsx`
   - Un archivo de estilos o Tailwind para estas vistas.

### Entregable Esperado:
Responde entregando el código completo (React/JSX y CSS) de estas vistas, listo para que el usuario pueda copiarlo y probarlo en su servidor `npm run dev`.
