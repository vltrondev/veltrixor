# Tarea 13: Internacionalización (Inglés) y Tipografía Moderna

**Asignado a:** Agente Especialista en Frontend (React & Vite)
**Estado:** Pendiente

## Descripción de la Tarea (Emergente)
El mercado principal de Veltrixor, LaborSignal y PriceTrackr está enfocado exclusivamente en **Estados Unidos**. Por lo tanto, toda la aplicación debe estar en **inglés**. Además, necesitamos actualizar la tipografía global para darle un toque más moderno, limpio y tecnológico.

### Requerimientos Estrictos:
1. **Traducción Total**: Revisa todos los componentes React (`.jsx`, `.tsx`, `.html`) y cambia cualquier texto local estático de español a **inglés fluido y corporativo**.
   - Por ejemplo: "Inicio" -> "Home", "Iniciar Sesión" -> "Sign In", "Registrarse" -> "Sign Up".
2. **Tipografía Moderna**: Importa y configura una fuente web moderna de Google Fonts (recomendaciones: *Inter*, *Outfit*, o *Plus Jakarta Sans*).
3. **Aplicación Global**: Actualiza tu `index.css` o el archivo de variables globales para que la fuente `font-family` se aplique por defecto a todo el documento `body`, los botones y los modales. Asegúrate de que los pesajes (weights) sean correctos (400 regular, 500 medium, 600 semi-bold o 700 bold).

### Entregable Esperado:
- El código actualizado donde se evidencie el cambio de los textos al idioma inglés.
- El archivo CSS global (o configuración de Tailwind) con la nueva tipografía importada y asignada.
